This assignment is about administering a National Geographic Quiz for students.
The quiz is a multiple choice based ie, a question followed by 4 choices from which only one is a the correct answer

Your program will read questions from the file:
/gaia/class/student/srivatss/IPC/ngbee/questionbank.txt

The file contains 30 questions,  each question is 6 lines long.  The first line is the question, followed by 4 choices.  The sixth line is the correct answer that should be matched to the response the user gives.

Prompt the five questions to the student:

After each question, the four choices are prompted. 
The student chooses one correct answer by typing 1, 2, 3 or 4 followed by <enter>
When student enters the number, you check if the entered number matches the correct answer
if the answer is correct, increment the score
The maximum score a student can get is 5/5.
